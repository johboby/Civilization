## Overview

This is a multi-agent civilization evolution simulation platform that combines artificial intelligence, complex systems theory, and game theory to simulate the development and interaction of multiple civilizations.

### Key Features

- **Multi-Agent System**: Independent civilizations with unique behaviors
- **Technology Tree**: 16 technologies across 4 levels
- **Strategic Decision Making**: 7 strategy types with dynamic adaptation
- **Resource Management**: Territory and resource simulation
- **Advanced AI**: Game theory and metacognitive learning
- **Rich Visualization**: Multiple chart types and data export
